# 👨🏻‍💻 Customer Behavior Analytics Project

This project is a complete end-to-end Data Analytics portfolio project focused on analyzing customer shopping behavior using Python, SQL, and Power BI. The goal of this project is to simulate a real-world business analytics workflow — from raw data processing to dashboard creation and business insight generation.

The project demonstrates practical skills in:
- Data Cleaning & Exploration
- SQL-based Business Analysis
- Data Visualization & Dashboarding
- Business Insight Generation
- Reporting & Presentation

It is designed to showcase my hands-on experience in data analytics and business intelligence tools.

---

# 📌 Project Overview

In this project, I worked on customer shopping behavior data to uncover patterns related to:
- Customer purchasing trends
- Product preferences
- Spending behavior
- Customer segmentation
- Sales insights
- Loyalty and retention indicators

The workflow covers the complete analytics lifecycle:

✅ Data Cleaning & Preparation using Python  
✅ Exploratory Data Analysis (EDA)  
✅ SQL Queries for Business Insights  
✅ Interactive Dashboard Creation in Power BI  
✅ Final Reporting & Presentation of Findings

# 🛠️ Tech Stack

# 📂 Project Workflow

## 1️⃣ Data Collection & Understanding
- Imported customer behavior dataset
- Understood columns, missing values, and data structure
- Performed initial exploration

## 2️⃣ Data Cleaning & Preprocessing
- Handled missing values
- Removed inconsistencies
- Standardized data formats
- Prepared dataset for SQL and visualization

## 3️⃣ Exploratory Data Analysis (EDA)
Performed analysis to identify:
- Customer spending patterns
- Popular product categories
- Seasonal trends
- High-value customer groups
- Purchase frequency behavior

## 4️⃣ SQL Analysis
Used SQL queries to solve business-related questions such as:
- Top-performing product categories
- Customer segmentation
- Revenue analysis
- Purchase behavior trends
- Repeat customer identification

## 5️⃣ Power BI Dashboard
Created an interactive dashboard with:
- KPI cards
- Sales trends
- Customer insights
- Product performance charts
- Filters and slicers for dynamic analysis

## 6️⃣ Final Insights & Reporting
Generated business insights and recommendations based on the analysis to support data-driven decision making.

# 🚀 How to Run the Project

## Clone the Repository
```bash
git clone <your-repository-link>
cd <repository-folder>
```

## Open the Jupyter Notebook
Run:
```bash
Customer_Shopping_Behavior_Analysis.ipynb
```

## Database Setup
- Create a database in PostgreSQL/MySQL
- Load the cleaned dataset into SQL
- Run SQL queries from:
```bash
customer_behavior_sql_queries.sql
```

## Open Power BI Dashboard
Open:
```bash
customer_behavior_dashboard.pbix
```
